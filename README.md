# OMDB Movie Search Project

This is a simple movie search web application built with HTML, CSS and JavaScript.
aee49c3f

## Features

- Search movies by title
- Display movie poster, title, year, runtime and type
- Show IMDb rating, genre, director, actors, language and plot
- Error handling for empty input and not found movies
- Loading message while searching
- Saves the last searched movie using localStorage
- Responsive design for desktop and mobile devices

## Technologies Used

- HTML
- CSS
- JavaScript
- OMDB API

## How to Use

1. Enter a movie name in the search input.
2. Click the Search button.
3. Movie details will be displayed on the page.

## Project Purpose

This project was created as part of a frontend development application task. The goal is to demonstrate basic API usage, DOM manipulation, responsive design and clean project structure using only HTML, CSS and JavaScript.
